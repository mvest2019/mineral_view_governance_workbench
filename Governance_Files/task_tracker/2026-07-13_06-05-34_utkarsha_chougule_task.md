Task Title: Mobile Responsiveness for Field Reports (Lease Report, Reservoir Report & Well Report)

Task Description:

Perform comprehensive mobile responsiveness testing for the Field Reports module to ensure that all report pages are fully functional, visually consistent, and user-friendly across mobile devices and smaller screen resolutions.

This task includes validating the following report types:

- Lease Report
- Reservoir Report
- Well Report

Scope of Testing

1. Lease Report

- Verify the Download button is visible on mobile screens.
- Ensure the Download button is properly aligned and does not overlap with the Selected Owner section.
- Validate that the Download functionality works correctly after tapping the button.
- Check that lease details such as Operator, Lease Status, Acres, Location, Play Type, Field Name, and Production Date are readable.
- Verify statistic cards (EUR, Produced, Reserves, Oil Report, Gas Report) stack correctly on smaller screens.
- Ensure charts resize properly and remain readable.
- Validate that filter controls and dropdowns are accessible on mobile.
- Verify tables support horizontal scrolling where required.
- Ensure the map section loads correctly, remains interactive, and does not break the layout.
- Check that all buttons such as View Owners and View Full Chart are tappable.

2. Reservoir Report

- Verify reservoir tabs/chips display properly on mobile.
- Validate reservoir information including status, performance, well counts, and production dates.
- Ensure all production statistic cards are responsive.
- Verify production charts render correctly without clipping or overlap.
- Validate timeline filters and dropdown selections on touch devices.
- Check the Reservoir Comparison chart and Reservoir Performance Metrics section for proper alignment and readability.
- Ensure tables can be scrolled horizontally when necessary.

3. Well Report

- Verify well number selection chips wrap correctly on small screens.
- Validate well details such as API No, Tracking No, Field Name, Play Type, Well Type, and Wellbore Profile.
- Ensure production test data, depth, and perforation cards display properly without overlapping.
- Verify the Reservoir section is responsive.
- Validate Oil and Gas production charts on mobile devices.
- Check the Formation Tops table for readability and horizontal scrolling support.
- Verify the Attachments section wraps correctly into multiple rows and all attachment links remain accessible.
- Ensure all export and action buttons are visible and functional.

Responsive Validation Checklist

- No unwanted horizontal scrolling on the page.
- No overlapping UI elements.
- No truncated or hidden text.
- Cards stack properly on smaller screens.
- Charts resize correctly and remain readable.
- Tables are scrollable without breaking the layout.
- Maps remain interactive.
- Dropdowns, filters, and buttons are touch-friendly.
- Consistent spacing, alignment, and typography across all reports.
- Lease Report Download option is available and working on mobile.

Devices to Test

- iPhone SE
- iPhone 12/13/14
- Samsung Galaxy S21/S22
- Google Pixel 7/8
- iPad Mini / iPad Air (optional)

Expected Result

All three Field Reports (Lease Report, Reservoir Report, and Well Report) should be fully responsive on mobile devices. Users should be able to view all information, interact with charts, tables, maps, filters, and attachments without layout issues. The Download option in the Lease Report must be clearly visible and function correctly on mobile screens.
