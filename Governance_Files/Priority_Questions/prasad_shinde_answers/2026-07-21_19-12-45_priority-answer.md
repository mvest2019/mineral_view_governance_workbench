# Priority Question Answer

Question:
Does the Campaigns report page expose mineral-owner PII, and is it gated by role or subscription tier?

Answer:
Yes, it shows PII — recipient names/emails and who replied/bounced/unsubscribed.
Gating: authentication only. The campaigns API (/api/campaigns, /api/campaigns/[id]) checks for a valid session but has no role check, and the proxy doesn't restrict /campaigns by role. So any logged-in CRM user (Admin/Marketing/Analyst) can view it. Subscription tier is a members-DB concept and doesn't apply to CRM-user access.
