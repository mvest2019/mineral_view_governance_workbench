# Priority Question Answer

Question:
Are the Microsoft Clarity session recordings masked, and do they capture owner PII or logged-in paid-user sessions?

Answer:
Yes. Microsoft Clarity is configured to mask personally identifiable information (PII), helping to ensure that sensitive user data is not exposed in session recordings.

Clarity captures user sessions whenever visitors interact with the website, including navigation, clicks, scrolling, and other on-page interactions. This can include sessions from visitors, registered users, and logged-in users if they interact with pages where Clarity is enabled. The recordings are used to understand user behavior, identify usability issues, and improve the overall website experience.

Although user sessions are recorded for behavioral analysis, sensitive personal information remains protected through Clarity's PII masking features.
