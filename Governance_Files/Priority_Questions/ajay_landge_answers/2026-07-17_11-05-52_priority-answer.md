# Priority Question Answer

Question:
Do the published BOLD and Mineral View reels need a no-advice / accuracy disclaimer before going live?

Answer:
For the content itself — yes, it's worth treating both as needing a no-advice / accuracy line, and we already lean that way: the BOLD metals format was built with compliance rules baked in, and the CTAs default to engagement rather than pushing a financial action. Both brands sit near sensitive ground (BOLD talks precious-metals prices, Mineral View talks mineral owners/leases), so anything that could read as "buy/sell" or a return promise is where a disclaimer matters most.

Practical take: if a reel states prices, ratios, or anything that could be taken as investment/financial guidance, a short "for information only, not financial advice — figures as of [date], verify before acting" style line is the safe default before it goes live. Purely brand/educational reels with no numbers or recommendations are lower-risk.
