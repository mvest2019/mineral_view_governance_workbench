# Priority Question Answer

Question:
Where are the credentials/OAuth tokens for the multiple social media accounts stored?

Answer:
Platform/API keys (e.g. LLM provider keys) — stored in the app's Settings/database and also in .env, with a known gotcha that a stale .env key can shadow the funded DB key.
Deployment secrets — production config lives in Vercel environment variables (e.g. the backend URL is re-pointed there when it rotates).
Meta (IG/FB) access — flows through our own Meta app ("SM Agentic Workflow") via the Graph API, so the token/app-secret sits with that Meta developer app.
