# Priority Question Answer

Question:
Is Trendelier an internal marketing tool or a customer-facing product, and which repo governs it?

Answer:
What it is: an internal marketing tool — we use it in-house to run our brand channels (scanning sources, generating insights/drafts, tracking analytics). It's not a customer-facing product; there are no external users or public sign-ups. The only thing that reaches the public is the finished content we choose to publish to the brand accounts after review.

Repo: it's governed by the Trendelier/trendelier repo, with the workflow being branch → push → PR → merge to main (never pushing straight to main), and Vercel auto-deploys from main to production.
