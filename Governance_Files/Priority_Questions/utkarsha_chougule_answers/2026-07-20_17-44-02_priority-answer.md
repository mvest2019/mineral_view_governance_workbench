# Priority Question Answer

Question:
What happens to a user's claimed leases and gated features when a subscription is downgraded or a yearly plan lapses?

Answer:
If a user downgrades their subscription, the downgrade takes effect at the end of the current billing period, after which the user is moved to the Free plan. Similarly, when a yearly subscription expires and is not renewed, the user is automatically moved to the Free plan, and access to paid features is removed.
