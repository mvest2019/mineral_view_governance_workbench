# Priority Question Answer

Question:
Are Field Reports and the Lease Report Download gated by subscription tier, and which tier will be used for testing?

Answer:
Field Reports and the Lease Report Download are not gated by subscription tier for downloading. Subscription tiers only determine which owners or leases a user can access. Any lease data that is visible and accessible within the selected account should also be available in the corresponding report and can be downloaded. Testing will be performed using an account that has access to the relevant lease data, and the downloaded report is expected to match the lease information displayed in the application.
